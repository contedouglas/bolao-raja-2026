# Bolão Raja 2026

Atualização automática da classificação do Brasileirão via GitHub Actions.

## Como funciona

- GitHub Actions roda a cada hora (10h-23h horário de Brasília)
- Faz scraping do site da UFMG
- Salva no Supabase
